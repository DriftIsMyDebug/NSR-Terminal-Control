# NSR-Terminal-Control
Система автоматизации логистики для терминалов порта Новороссийск.

## Цели проекта (Марафон до 5 августа):
- Реализация системы управления очередями (Java Core).
- Имитация работы КПП и весовых платформ.
- Интеграция с БД PostgreSQL.
- Создание REST API на Spring Boot.
